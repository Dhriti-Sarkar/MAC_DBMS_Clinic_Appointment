# Clinic Appointment System (DBMS Project)

This is a full-stack Clinic Appointment Management System built as part of a DBMS course project.

## Tech Stack
- MariaDB
- Node.js (Express)
- HTML, CSS, JavaScript

## Features
- Add patients (name, gender, phone)
- View doctors
- Book appointments
- View appointment schedule
- Delete patients (with foreign key constraint handling)
- Delete appointments

## How to Run the Project

### 1. Start Backend
```bash
cd backend
node server.js
